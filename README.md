# Unwyze

Unwyze is a third-party, fully local iOS app for the [Wyze Scale S](https://www.wyze.com/products/wyze-scale-s) with HealthKit support. It uses a reverse-engineered implementation of the protocol needed to talk to the scale.

> [!WARNING]  
> [The protocol](PROTOCOL.md) that is used here was largely reversed by an LLM. It seems to work but obviously this might break things or cause issues.
